# AI 财富自由计划 · 计划库（商业计划书级）

你的「AI 财富自由」作战室：每小时自动头脑风暴产出新计划（商业计划书级），页面内可分类、对比 AI 初评、自己评分备注。

## 🔗 固定链接（公开 Pages，已上线）
**https://tiger-hz.github.io/ai-wealth-plans/**
> 说明：该 GitHub 账户为免费计划，仅支持公开仓库的 Pages。按你的选择，本页为**公开可见**（链接任何人可访问）。计划内容对外可见；但**你的评分/备注仅存浏览器本地，不会上传**。

## 文件说明
| 文件 | 作用 |
|------|------|
| `index.html` | **交互主页面**：分类筛选、AI初评（只读）、我的可行性/优先级评分、备注、统计、导入导出。 |
| `plans-data.js` | 由 `plans.json` 自动生成，页面通过 `<script>` 加载。 |
| `plans.json` | 规范数据源（商业计划书级字段）。 |
| `planlib.py` | 数据底座：`init` 重建、`add <batch.json>` 追加新批次。 |
| `daily_digest.py` | 每日推送：汇总当日新增 → `每日推送.md`。 |
| `头脑风暴计划库.md` | 人类可读全量计划（商业计划书级）。 |
| `每日推送.md` | 每日自动汇总。 |

## 商业计划书级字段（每条计划）
一句话定位 · 痛点与机会 · 产品与功能（PM+AI视角） · 目标市场与受众（TAM/SAM/SOM） · 商业模式与变现 · 预期投入 · 预期收益 · 现有市场与竞品 · 关键挑战与风险 · 合规边界 · 进度计划 · AI初评（可行性/优先级/建议）。

## 自动化任务（已在调度服务中配置）
- **每小时**：头脑风暴引擎产出 ≥10 个多元赛道计划（商业计划书级），自动入库并 `git push`。
- **每天 ~21:00**：生成当日推送，汇总到 `每日推送.md` 并推送。
- 仓库每次推送后，GitHub Pages 会自动重新构建，页面随之更新。

## 如何查看
- 在线：直接打开上面的 Pages 链接（公开）。
- 本地：双击 `index.html`（完全离线，评分存本地）。
- 跨设备同步评分：页面「导出我的评分」→ 另一设备「导入」。

## 如需改回「仅自己可见」
免费账户不支持私有仓库 Pages。可选：① 升级 GitHub Pro 后改回私有并开启私有 Pages；② 仅用私有仓库 + 本地打开（最私密）。
