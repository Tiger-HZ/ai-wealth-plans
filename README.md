# AI 财富自由计划 · 私有计划库

你的私人「AI 财富自由」作战室：每小时自动头脑风暴产出新计划，页面内可分类、评分、备注。

## 文件说明
| 文件 | 作用 |
|------|------|
| `index.html` | **交互主页面**：分类筛选、可行性/优先级评分、备注、统计、导入导出。双击即可打开。 |
| `plans-data.js` | 由 `plans.json` 自动生成，页面通过 `<script>` 加载（file:// 也能用，无跨域问题）。 |
| `plans.json` | 规范数据源（所有计划的数组）。 |
| `planlib.py` | 数据底座脚本：`init` 重建、`add <batch.json>` 追加新批次。 |
| `daily_digest.py` | 每日推送脚本：汇总当日新增并写到 `每日推送.md`。 |
| `头脑风暴计划库.md` | 人类可读的全量计划日志。 |
| `每日推送.md` | 每日自动汇总。 |
| `deploy.sh` | 一键部署到**私有** GitHub Pages（仅你可见）。 |

## 如何查看（最简单）
直接双击 `index.html` 用浏览器打开即可。所有评分/备注存在你浏览器本地（localStorage），**不上传、不外传**。

## 自动化任务（已在调度服务中配置）
- **每小时**：头脑风暴引擎产出 ≥10 个新计划（覆盖专业/文学/心理/金融/INFJ社群等多元赛道），自动写入 `plans.json` 并刷新页面。
- **每天 ~21:00**：生成当日推送，汇总到 `每日推送.md`。

## 部署为固定私有链接（可选）
1. 终端执行 `gh auth login` 完成 GitHub 登录；
2. 在本目录执行 `bash deploy.sh`；
3. 打开 `https://<你的账号>.github.io/ai-wealth-plans/`（仅登录该账号可见）。

## 隐私说明
- 计划内容存放于你的仓库/本地，评分备注仅存浏览器本地。
- 如需跨设备同步评分，使用页面「导出我的评分」下载 JSON，在另一设备「导入」。
