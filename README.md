# AI 财富自由计划 · 私有计划库（商业计划书级）

你的私人「AI 财富自由」作战室：每小时自动头脑风暴产出新计划（商业计划书级），页面内可分类、AI初评对比、自己评分备注。

## 文件说明
| 文件 | 作用 |
|------|------|
| `index.html` | **交互主页面**：分类筛选、AI初评（只读）、我的可行性/优先级评分、备注、统计、导入导出。双击即可打开。 |
| `plans-data.js` | 由 `plans.json` 自动生成，页面通过 `<script>` 加载（file:// 也能用）。 |
| `plans.json` | 规范数据源（商业计划书级字段：定位/痛点/功能/市场/商业模式/投入/收益/竞品/挑战/合规/进度/AI初评）。 |
| `planlib.py` | 数据底座：`init` 重建、`add <batch.json>` 追加新批次。 |
| `daily_digest.py` | 每日推送：汇总当日新增 → `每日推送.md`。 |
| `头脑风暴计划库.md` | 人类可读全量计划（商业计划书级）。 |
| `每日推送.md` | 每日自动汇总。 |

## 商业计划书级字段（每条计划）
一句话定位 · 痛点与机会 · 产品与功能（PM+AI视角） · 目标市场与受众（TAM/SAM/SOM） · 商业模式与变现 · 预期投入 · 预期收益 · 现有市场与竞品 · 关键挑战与风险 · 合规边界 · 进度计划 · AI初评（可行性/优先级/建议）。

## 如何查看（最简单，完全私有）
直接双击 `index.html`。所有**评分/备注存在你浏览器本地（localStorage），不上传、不外传**。

## 自动化任务（已在调度服务中配置）
- **每小时**：头脑风暴引擎产出 ≥10 个多元赛道计划（商业计划书级），自动入库并 `git push` 到私有仓库。
- **每天 ~21:00**：生成当日推送，汇总到 `每日推送.md` 并推送。
